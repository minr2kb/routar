# @routar/ky

ky-based executor for [routar](https://github.com/minr2kb/routar) — transport adapter using [ky](https://github.com/sindresorhus/ky).

## Install

```bash
npm install @routar/core @routar/ky ky
```

See the [documentation](https://routar.vercel.app) or the [main README](https://github.com/minr2kb/routar) for full documentation.
