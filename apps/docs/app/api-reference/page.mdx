# API Reference

## `@routar/core`

| Export | Description |
|--------|-------------|
| [`endpoint()`](/api-reference/endpoint) | Define a single typed endpoint |
| [`defineRouter()`](/api-reference/define-router) | Group endpoints under a URL prefix |
| [`createApi()`](/api-reference/create-api) | Build a typed API client from executor + router |
| [`createExecutor()`](/api-reference/create-executor) | Wrap a transport function with middleware |
| [`withRetry`](/api-reference/middleware#withretry) | Retry middleware |
| [`withTimeout`](/api-reference/middleware#withtimeout) | Timeout middleware |
| [`withLogger`](/api-reference/middleware#withlogger) | Logger middleware |
| [`defineMiddleware`](/api-reference/middleware#definemiddleware) | Create custom middleware |
| `ValidationError` | Thrown when request/response validation fails |
| `ApiTypes<TApi>` | Utility type to extract request/response types |
| `PathParams<TPath>` | Utility type to extract `:param` names from a path |
