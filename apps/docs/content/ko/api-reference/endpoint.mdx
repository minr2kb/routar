# `endpoint(spec)`

단일 엔드포인트를 정의합니다. `adapter`에서 완전한 타입 추론을 제공합니다.

```ts
import { endpoint } from '@routar/core'
import { z } from 'zod'

const TodoSchema = z.object({ id: z.number(), title: z.string() })

endpoint({ method: 'GET', path: '/', response: z.array(TodoSchema) })

endpoint({
  method: 'GET',
  path: '/',
  response: z.array(TodoSchema),
  adapter: (raw) => raw.map(todo => ({ ...todo, label: todo.title })),
})
```

## 파라미터

| 필드 | 타입 | 필수 | 설명 |
|-------|------|----------|-------------|
| `method` | `'GET' \| 'POST' \| 'PUT' \| 'PATCH' \| 'DELETE'` | Yes | HTTP 메서드 |
| `path` | `string` | Yes | 라우터 접두사에 상대적인 경로 |
| `response` | `AnyValidator<T>` | Yes | 응답 검증 스키마. `.parse(data): T`를 가진 객체 또는 [Standard Schema](#standard-schema) |
| `request` | `{ path?, query?, body? }` | No | 요청 스키마 — 각 부분을 개별 validator로 담은 [버킷 맵](#요청-버킷). `path`에 `:param`이 있으면 `request.path`가 필수입니다. |
| `adapter` | `(raw: T) => U` | No | 검증된 응답 변환 |

## 요청 버킷

`request`는 각 부분의 개별 validator를 담은 `{ path?, query?, body? }` 맵입니다 — 부분당 validator 하나. routar가 버킷을 단일 내부 envelope로 합성하므로 호출부·query key·react-query flatten·MSW 모두 동일한 `{ path, query, body }` shape을 봅니다.

```ts
endpoint({
  method: 'GET',
  path: '/:id',
  request: {
    path: z.object({ id: z.number() }),
    query: z.object({ q: z.string() }),
  },
  response: TodoSchema,
})
```

Path param 강제는 그대로 적용됩니다: `path`에 `:param`이 있으면 `request.path`가 필수이며 해당 키를 포함해야 합니다.

## Standard Schema

`request`/`response`(및 각 버킷)는 `.parse()`를 가진 객체 외에도 [Standard Schema](https://standardschema.dev)(`~standard` 인터페이스 — ArkType, Zod 3.24+, Valibot 등)를 받습니다. 런타임은 `.parse()`가 있으면 우선 사용하고 없으면 `~standard.validate`(sync/async)를 호출합니다. Standard Schema 이슈는 `StandardSchemaError`로 변환되어 `ValidationError`로 래핑됩니다.

```ts
import { type } from 'arktype'

endpoint({
  method: 'GET',
  path: '/',
  response: type({ id: 'number', title: 'string' }).array(), // ArkType — .parse 없음
})
```

## Path param 강제

```ts
// ✅ 올바름
endpoint({ path: '/:id', request: { path: z.object({ id: z.number() }) }, response: TodoSchema, method: 'GET' })

// ❌ 컴파일 에러
endpoint({ path: '/:id', request: { query: z.object({ q: z.string() }) }, response: TodoSchema, method: 'GET' })
```

## Adapter

```ts
const TodoRawSchema = z.object({ id: z.number(), title: z.string() })

endpoint({
  method: 'GET', path: '/',
  response: z.array(TodoRawSchema),
  adapter: (raw) => raw.map(r => ({ ...r, label: r.title })),
})
```

`response` 스키마에 `.transform()`을 사용하지 마세요.

## 타입 유틸리티

```ts
import type { ApiTypes, PathParams } from '@routar/core'

type TodoApiTypes = ApiTypes<typeof todoApi>
type P = PathParams<'/:userId/posts/:postId'> // 'userId' | 'postId'
```
