# 플러그인

플러그인은 선택적 라이프사이클 훅을 가진 이름 있는 객체입니다. `createExecutor`의 `plugins` 옵션으로 전달합니다. `retry`와 `timeout`은 `createFetchExecutor`의 first-class 옵션입니다.

```ts
import { createExecutor, definePlugin, logger } from '@routar/core'

const authPlugin = definePlugin({
  name: 'auth',
  onRequest: async (opts) => ({
    ...opts,
    headers: { ...opts.headers, Authorization: `Bearer ${await getToken()}` },
  }),
  onError: async (err) => {
    if (isUnauthorized(err)) await refreshToken()
    throw err
  },
})

const executor = createExecutor(transport, {
  plugins: [authPlugin, logger({ log: (msg, data) => myLogger.debug(msg, data) })],
})
```

## 플러그인 인터페이스

```ts
interface ExecutorPlugin {
  name?: string
  onRequest?: (opts: ExecuteOptions) => ExecuteOptions | Promise<ExecuteOptions>
  onResponse?: (response: unknown, opts: ExecuteOptions) => unknown | Promise<unknown>
  onError?: (error: unknown, opts: ExecuteOptions) => never | Promise<never>
}
```

모든 훅은 선택적입니다. 필요한 것만 구현하세요.

## 내장 플러그인

### `logger(options?)`

메서드, URL, 소요 시간을 로깅합니다.

| 옵션 | 타입 | 기본값 | 설명 |
|------|------|--------|------|
| `log` | `(msg: string, data?: unknown) => void` | `console.log` | 커스텀 로그 함수 |

```ts
logger()
logger({ log: (msg, data) => myLogger.debug(msg, data) })
```

## 커스텀 플러그인

`definePlugin`으로 타입 추론을 제공합니다:

```ts
import { definePlugin } from '@routar/core'

const correlationPlugin = definePlugin({
  name: 'correlation-id',
  onRequest: (opts) => ({
    ...opts,
    headers: { ...opts.headers, 'X-Request-Id': crypto.randomUUID() },
  }),
})

const errorNormalizerPlugin = definePlugin({
  onError: async (err) => {
    if (isAxiosError(err)) throw new HttpError(err.response?.status ?? 0, err.message)
    throw err
  },
})
```

## retry와 timeout

`retry`와 `timeout`은 `createFetchExecutor`에서 사용합니다:

```ts
import { createFetchExecutor } from '@routar/core'

const executor = createFetchExecutor('https://api.example.com', {
  retry: 3,
  timeout: 5_000,
})
```

`retry`는 숫자 또는 `shouldRetry` 옵션 객체를 받습니다:

```ts
retry: { count: 3, shouldRetry: (err) => !(err instanceof HttpError && err.status < 500) }
```

타임아웃 발생 시 `TimeoutError`가 던져집니다:

```ts
import { TimeoutError } from '@routar/core'

try {
  await api.getList()
} catch (err) {
  if (err instanceof TimeoutError) {
    console.log(`${err.ms}ms 후 타임아웃`)
  }
}
```

axios와 ky는 각 라이브러리 인스턴스에서 설정하세요 (`axios.create({ timeout })`, `ky.create({ timeout, retry })`).
