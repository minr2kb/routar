# API Reference

## `@routar/core`

| Export | Description |
|--------|-------------|
| [`endpoint()`](/en/api-reference/endpoint) | Define a single typed endpoint |
| [`defineRouter()`](/en/api-reference/define-router) | Group endpoints under a URL prefix |
| [`createApi()`](/en/api-reference/create-api) | Build a typed API client from executor + router |
| [`createExecutor()`](/en/api-reference/create-executor) | Wrap a transport function with plugins |
| [`dispatchExecutor()`](/en/api-reference/dispatch-executor) | Select a transport per request at runtime |
| [`createFetchExecutor()`](/en/executors/fetch) | Built-in fetch executor |
| `HttpError` | Thrown by `createFetchExecutor` on non-2xx responses |
| [`definePlugin`](/en/api-reference/plugins#custom-plugins) | Create a custom executor plugin |
| [`logger`](/en/api-reference/plugins#built-in-plugin) | Built-in logging plugin |
| [`TimeoutError`](/en/api-reference/plugins#retry-and-timeout) | Thrown when a `createFetchExecutor` timeout fires |
| `ValidationError` | Thrown when request/response validation fails |
| [`ApiTypes<TApi>`](/en/api-reference/api-types) | Utility type to extract request/response types from an API client |
| `PathParams<TPath>` | Utility type to extract `:param` names from a path |
